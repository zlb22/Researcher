# 伯克希尔·哈撒韦 (Berkshire Hathaway) 2019-2023年财务数据

## 公司概况
伯克希尔·哈撒韦是由沃伦·巴菲特领导的多元化投资集团，保险业务是其核心业务之一。

## 关键财务指标趋势
1. **营收强劲增长**: 营收持续增长，反映业务扩张
2. **净利润波动**: 受投资市场影响较大，2022年出现亏损
3. **资产规模扩大**: 总资产突破1万亿美元
4. **保险业务稳健**: 保险浮存金持续增长
5. **ROE恢复**: 2023年ROE达到14.2%，恢复强劲盈利能力

## 业务构成分析
1. **保险业务**: GEICO、伯克希尔再保险等
2. **铁路业务**: BNSF铁路
3. **能源业务**: 伯克希尔能源
4. **制造业**: 精密铸件等
5. **投资组合**: 苹果、美国银行等大量持股

## 数据备注
- 净利润包含投资损益，波动较大
- 保险浮存金是保险业务的重要指标
- 2022年亏损主要由于投资组合价值下降
- 营收增长主要来自保险和能源业务
