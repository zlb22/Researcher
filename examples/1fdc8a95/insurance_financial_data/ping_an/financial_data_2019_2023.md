## 关键财务指标趋势
1. **营收调整**: 2021-2023年营收有所调整
2. **净利润下降**: 受宏观经济和投资环境影响
3. **资产规模增长**: 总资产突破10万亿元
4. **ROE下降**: 从高点的24.4%下降到10.5%左右
5. **保费收入稳定**: 保费收入保持相对稳定

## 业务板块分析
1. **寿险及健康险**: 平安寿险、平安健康险
2. **财产保险**: 平安产险
3. **银行业务**: 平安银行
4. **资产管理**: 平安资产管理
5. **科技业务**: 金融壹账通、平安好医生等

## 战略转型
1. **金融+科技**: 推动数字化转型
2. **金融+生态**: 构建医疗健康生态圈
3. **综合金融**: 交叉销售和客户迁徙
4. **可持续发展**: ESG和绿色金融

## 市场挑战
1. **利率下行**: 影响保险资金投资收益
2. **监管变化**: 保险产品监管趋严
3. **竞争加剧**: 互联网保险冲击
4. **经济环境**: 宏观经济增速放缓

## 数据备注
- 数据单位为人民币
- 净利润为归属于母公司股东的净利润
- ROE为加权平均净资产收益率
- 保费收入为原保险保费收入
