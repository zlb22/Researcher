# 保险公司融资与资本结构研究框架

## 研究目标
收集以下6家保险公司的融资情况和资本结构信息（2020-2024年）：
1. 安联集团 (Allianz)
2. 伯克希尔·哈撒韦 (Berkshire Hathaway)
3. 平安保险 (Ping An Insurance)
4. 安盛集团 (AXA)
5. 联合健康集团 (UnitedHealth Group)
6. 中国人寿 (China Life Insurance)

## 需要收集的信息维度
1. 债务水平和债务结构
2. 股权融资历史
3. 债券发行情况
4. 资本充足率（偿付能力充足率）
5. 现金储备和流动性状况
6. 融资成本（债券利率等）

## 数据来源计划
- 公司年度报告（2020-2023）
- 季度财务报告
- 债券发行公告
- 信用评级报告
- 监管机构披露
- 财经新闻和分析报告

## 文件结构
```
insurance_capital_structure/

└── china_life/       # 中国人寿数据
```

## 时间范围
重点关注2020-2024年数据，特别是2023-2024年最新信息。

## 数据质量要求
- 优先使用官方财务报告
- 注明数据来源和日期
- 区分历史数据和最新数据
- 注意货币单位（美元、欧元、人民币）
